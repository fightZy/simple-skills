# Archive Summary Template

Use for compressed historical summaries under `summaries/archive.md` or split archive files.

Metadata should follow [../schema.md](../schema.md).
Generated archive files should prefer script-owned frontmatter.
Archive summaries should maintain `source_ids` for the archived session entries they represent.

```md
# Archived Memory

## Important History
- Decision or milestone with one-line rationale

## Reusable Context
- Older knowledge that still matters
```

# Generated Recent Summary Template

Use for script-generated current-state summaries such as `summaries/recent.md`.

Metadata should be generated by scripts such as `record_session.py` or `refine_memory.py`.
Read [../schema.md](../schema.md) for field expectations.
Generated recent summaries should maintain `source_ids` for the session entries they currently represent.

```md
# Recent Memory

## Active Context
- Current priorities
- Current risks

## Recent Sessions
- <YYYY-MM-DD> [topic]: one-line summary and next step

## Pending Follow-ups
- Item
```

# Crystal Template

Use for durable knowledge that should influence future work.

Metadata should follow [../schema.md](../schema.md).
When crystal creation is scripted later, the script should own frontmatter generation.

```md
# Crystal: repo-local markdown first

## Statement
The durable rule, decision, pattern, or insight.

## Why It Matters
Why future contributors should care.

## When To Apply
Situations where this knowledge should guide actions.

## Provenance
- Source session or summary ids

## Notes
- Caveats, exceptions, or supersession notes
```
